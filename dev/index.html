ARCEMEAnalysis.arceme_s1_position — Method
arceme_s1_position(ds)

Analyses the sentinel 1 time stamps of the datasets and groups them into series of time stamps whose difference is always a multiple of 6 days, so we can assume that all images with the same group tag are retrieved from the same position. Store the groups in the dataset with the name s1_postion.

ARCEMEAnalysis.arceme_set_httpstore — Method

Sets the https store from which to retrieve the cubes. If no local preference is set, the default location is used: https://s3.waw3-2.cloudferro.com/swift/v1

arceme_validpairs(;batch="ARCEME-DC-6")

Get valid pairs of ARCEME events from the local path (if set with arcemesetlocalpath) or the http data store. Default httpstore is "https://s3.waw3-2.cloudferro.com/swift/v1". It can be reset with arcemesethttpstore.

ARCEMEAnalysis.boa — Method
boa(band; BOA_ADD_OFFSET = -1000, QUANTIFICATION_VALUE = 10000)

Compute the radiometric offsets for Sentinel 2 bands to get values at bottom of atmosphere. The default values are valid for data processed from baseline 04.00 (January 2022) onwards. The entire Sentinel-2 archive in CDSE has been reprocessed and is now available in baseline 05.xx, with consistent offset.
